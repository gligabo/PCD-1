# Prática em Ciência de Dados - SME0852

## Disciplina
Trabalho prático desenvolvido na disciplina de **Ciência de Dados**, organizado em diferentes eixos temáticos que cobrem as principais etapas de um projeto de análise de dados.

O objetivo é aplicar conceitos teóricos em um **dataset real**, conduzindo o processo completo desde a análise inicial dos dados até a construção e avaliação de modelos.

## Estrutura do Trabalho

O projeto está organizado nos seguintes eixos:

### Eixo 1 — Análise Exploratória de Dados e Diagnóstico Estatístico
- compreensão da estrutura do dataset  
- análise descritiva das variáveis  
- identificação de padrões, distribuições e possíveis anomalias  

### Eixo 2 — Preparação de Dados e Engenharia de Atributos
- limpeza de dados  
- tratamento de valores ausentes  
- transformação e criação de novas variáveis  

### Eixo 3 — Aprendizado Supervisionado
- definição de problema preditivo  
- treinamento de modelos de classificação ou regressão  

### Eixo 4 — Aprendizado Não Supervisionado
- aplicação de técnicas como clustering ou redução de dimensionalidade  
- análise de estruturas e agrupamentos nos dados  

### Eixo 5 — Validação, Interpretabilidade e Robustez de Modelos
- avaliação de desempenho  
- análise de interpretabilidade  
- verificação de robustez e generalização dos modelos
